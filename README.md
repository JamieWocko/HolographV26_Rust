Holograph V26 in Rust.

- Please not this isn't a perfect port - it runs, however there are a lot of bugs which unfortunately I never got around to fixing.
- You can find the matching DCR pack here - https://wocko-dev.org/holo_v26_dcrs.zip

- You can also find my PHPRetro MySQLi port on my repo page.

- I recommend using MariaDB, or an older version of MySQL Server - there seems to be some query errors on newer MySQL Server revisions - which once again I never got around to fixing.

- MySQL credentials are in config/holograph.toml
-  Go into the server/main folder directory and enter "cargo run" and it should build.

-  I will offer very limited support, but like I said this is pretty barebones at the moment.

  Thank you.
